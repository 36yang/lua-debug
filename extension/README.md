Lua Debug Adapter for Visual Studio Code
=========================================

## Requirements

* Lua 5.3, 5.4
* OS: Windows

## Feature

* Breakpoints
* Conditional breakpoints
* Hit Conditional breakpoints
* Step over, step in, step out
* Watches
* Evaluate expressions
* Remote debugging
* Exception
